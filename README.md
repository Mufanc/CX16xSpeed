## TamperMonkey

## 我是小白

&emsp;&emsp;如果你是电脑小白，不懂得如何安装扩展程序，也可以使用我封装好的「套壳浏览器」，注意此程序需要依赖于你电脑中的 Chrome 或 **新版**Edge 浏览器才能正常运行，如果你双击之后没反应，请检查你的浏览器是否符合要求。
